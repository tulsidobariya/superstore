# 📊 Superstore Sales Analysis Dashboard (Power BI)

## 📌 Project Overview
This project is an interactive Power BI dashboard created using the Superstore dataset. The dashboard helps analyze sales performance, profit, orders, customers, and product categories through visual reports.

The objective is to convert raw business data into meaningful insights that help in decision-making.

---

## 🎯 Objectives
- Analyze total Sales, Profit, and Orders
- Identify top-performing Categories and Sub-Categories
- Compare sales across different Regions and States
- Analyze monthly and yearly sales trends
- Understand customer segments
- Build an interactive business dashboard

---

## 🛠️ Tools & Technologies
- Power BI Desktop
- Microsoft Excel / CSV Dataset
- Data Cleaning
- Data Visualization
- DAX (Basic Measures)

---

## 📂 Dataset
- **Dataset Name:** Superstore Sales Dataset
- **Format:** CSV
- Contains information about:
  - Order Date
  - Customer
  - Region
  - State
  - Category
  - Sub-Category
  - Sales
  - Profit
  - Quantity
  - Discount

---

## 📈 Dashboard Features

- 💰 Total Sales KPI
- 📦 Total Orders
- 📊 Total Profit
- 🌍 Sales by Region
- 🗺️ Sales by State
- 📅 Monthly Sales Trend
- 🛍️ Sales by Category
- 📦 Sales by Sub-Category
- 👥 Customer Segment Analysis
- 🔍 Interactive Filters (Slicers)

---

## 📊 Key Insights

- Technology category generated the highest sales.
- Some states recorded high sales but low profit.
- Discounts significantly affected profitability.
- Consumer segment contributed the highest revenue.
- Monthly sales trends help identify seasonal demand.
